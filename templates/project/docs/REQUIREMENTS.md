# Requirements

## Core Question
What is the one thing this project must answer, build, or prove?

## Who this is for
- Researcher: what question does this advance? what does the field currently not know?
- Engineer: what system capability is missing or broken?
- Developer: what user problem does this solve? what behavior changes for whom?

## Constraints
- Technical: what are the hard limits (compute, latency, data, dependencies)?
- Scope: what is explicitly out of scope for this phase?
- Quality: what does "good enough" mean here -- correctness, reliability, reproducibility?
