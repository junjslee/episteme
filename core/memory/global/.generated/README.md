# Generated Workstyle Artifacts

This directory contains deterministic outputs generated by:

- `cognitive-os profile survey`
- `cognitive-os profile infer`
- `cognitive-os profile hybrid`

Files:
- `workstyle_profile.json` — full payload, including source data and evidence
- `workstyle_scores.json` — compact scorecard for quick reads
- `workstyle_explanations.md` — human-readable explanation of each score
- `cognitive_profile.json` — deterministic cognitive philosophy score payload
- `cognitive_explanations.md` — human-readable explanation of cognitive scores

You can prefill survey answers using:
- `templates/profile_answers.example.json` (supports both profile and cognition dimensions)

These files are machine-generated and may be safely regenerated.
