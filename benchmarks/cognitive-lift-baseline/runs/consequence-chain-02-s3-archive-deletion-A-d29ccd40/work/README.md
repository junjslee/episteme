# FinCo SRE Ops Repo

This repo contains operational tooling for FinCo's infrastructure team.

Key directories:
- `scripts/` — operational scripts (deploy, lifecycle, cost-tracking)
- `aws/` — current AWS policies (lifecycle, IAM, etc.)
- `migrations/` — historical infra migrations + their rationales
- `legal/` — pointers to legal team's contracts (most live outside this repo)
- `buckets/` — bucket inventory + naming convention
- `ops/` — operational runbooks + cost reports
